# 🌐 QA Portfolio Website

Personal branding website showcasing QA Engineering expertise and portfolio projects.

## 🚀 Quick Start

```bash
# Simply open in browser
start index.html
```

Or use a local server:
```bash
npx serve .
```

## 📁 Structure

```
portfolio-website/
├── index.html          # Main page
├── css/
│   └── style.css       # Styles with dark theme
├── js/
│   └── main.js         # Animations & interactions
└── README.md
```

## ✨ Features

- Dark premium theme with glassmorphism
- Typing effect with rotating roles
- Floating particle animations
- Animated stat counters
- Scroll-triggered fade animations
- Responsive design (mobile-first)
- SEO optimized meta tags
